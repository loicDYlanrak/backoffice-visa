# Visa Application - Spring Boot CRUD avec JSP

## Description
Application Spring Boot simple avec CRUD complet utilisant JSP, Bootstrap et MySQL.

## Technologies utilisées
- Spring Boot 3.2.4
- Spring Data JPA
- MySQL
- JSP / JSTL
- Bootstrap 5
- Maven

## Prérequis
- JDK 17+
- MySQL 8+
- Maven (ou utilisation du Maven Wrapper inclus)
